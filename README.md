# Strk. prog. Tutorium WiSe2025

Dieses Repository dient dazu dass ihr euch unsere bearbeiteten Aufgaben anschauen, sowie den aktuellsten Stand unseres Programmierprojektes herunterladen könnt.

- Zum Programmierprojekt: 
Wir programmieren ein Pokémon inspiriertes Spiel auf der konsole wo 2 Spieler mit ihren Studimons gegeneinander kämpfen können.
Das Projekt dient zur vertiefung der Programmierkenntnise in C für die Studierende und soll praktische Anwendungen davon wiederspiegeln.

- Hinweis zu Funktionsnamen: manche Funktionen wie z.B. scanf() werden von meiner IDE (Visual Studio 2022) nicht direkt unterstüzt weshalb diese bei meinem bereitgestellten Code scanf_s() heißt. In der Klausur bitte die in der Vorlesung gelernten Funktionsnamen verwenden!
